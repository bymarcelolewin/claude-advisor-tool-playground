Welcome to the Advisor Tool Playground

Created by iBuildWith.ai

A hands-on sandbox for exploring Anthropic's advisor tool, a beta feature that lets a cheap, fast executor model consult a smarter advisor model mid-inference, only when it's actually needed.

Running Opus end-to-end is expensive. Running Sonnet or Haiku end-to-end is cheap but weaker on hard problems. The advisor strategy pairs them: Sonnet/Haiku drives the task, and only escalates to Opus when it hits a decision it isn't sure about. The result is near-Opus reasoning at near-Sonnet cost.

Read more: The Advisor Strategy · API docs

What this playground does

The advisor tool is hard to reason about from code alone. Two things are invisible until you instrument them:

  • Did the advisor actually fire? If your prompt is trivial, the executor skips it and you won't see any benefit.
  • Was the extra cost worth it? Compared to running Sonnet solo or Opus solo on the same prompt.

This playground answers both. Every conversation turn is broken down into its individual API steps — executor → advisor call → executor continues — with full token counts, costs, and the raw content each step produced. Compare mode runs the same prompt through up to three paths in parallel (advisor tool vs. executor-solo vs. advisor-solo) so you can see cost, latency, and quality deltas side by side. An optional LLM-as-judge evaluator scores which branch actually produced the better answer on a four-dimension rubric.

How to use it

  1. Pick your models from the Config Models panel above the chat. The canonical pair is Sonnet executor + Opus advisor — it's the whole point of the feature.
  2. Pick a Mode. Start with "Advisor only" for normal use. Switch to one of the compare modes to run the same prompt against baselines in parallel.
  3. Send a prompt from the chat pane. The left side shows the replies. The right side shows every API step that produced them.
  4. Click any turn's "Evaluate" button in the trace pane (compare mode only) to run an LLM-as-judge scoring pass — opt-in per turn, so nothing extra runs automatically.

💡 Tip: If the advisor branch shows 0 advisor calls, the executor didn't escalate — your prompt was too simple for it. Try a substantive task that benefits from planning.
